# finder

Finder allows you to find entries in another file

Expected input file

```text
a
b
c
d
e
f
g
```

Expected find file

```text
aa
b
cc
dd
e
f
g
```

Expected output

```text
aa
cc
dd
```

Usage:

```bash
finder -find [column number] -raw [column number]
```
